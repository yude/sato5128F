# [yudete](https://github.com/yudete) / sato5128F
Convert sentences in Japanese into さとちゃん formatted

## Dependencies
* MeCab
* unidic-lite
* [mecab-ipadic-neologd](https://github.com/neologd/mecab-ipadic-neologd)
```
pip install python3-mecab unidic-lite
```
